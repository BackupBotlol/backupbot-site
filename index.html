Bot Features

BackupBot is designed to provide peace of mind with comprehensive backup solutions for your Discord server.

Scheduled Backups

Automated backups with customizable timezone, frequency, and component settings to fit your needs.

Comprehensive Backups

Back up channels, roles, emojis, stickers, and server assets all in one ZIP file or as separate ZIP files.

Server Data Protection

Keep your server safe with a reliable backup solution that preserves your server.

Flexible Configuration

Choose what to backup, how often, and where to send the files, Complete control over your server's backups.

Easy Recovery

When disaster strikes (or your stuff gets nuked), get back on track faster than ever with our backup files that make restoration simple.

Secure Delivery

Backups are delivered directly to your desired Discord log channel with cloud storage for larger files.

How To Use

Getting started with BackupBot is pretty easy, Follow these simple steps to secure your server.

1

Add the Bot to Your Server

Click the "Add to your Server" button at the top of this page to authorize BackupBot with the necessary permissions.

2

Configure Backup Settings

Use the /addserver command to set up a log channel, backup frequency, and timezone for your scheduled backups.

3

Customize Your Backup Components

Use /configurebackupcomponents to choose exactly what gets saved and not saved in your server backups.

4

Relax and pop a glass

BackupBot will automatically create backups of your server and send them to your desired log channel on schedule.

Available Commands

BackupBot offers a variety of commands to help you manage your server backups effectively.

/help

Display all available commands and their descriptions

/addserver

Add your server to the backup list and set the log channel where backup files will be sent and the timezone and frequency

/activate

Activate the backup scheduler and run an immediate backup

/deactivate

Deactivate the backup scheduler for your server

/status

Check if the backup scheduler is active and when the next backup is scheduled

/removeserver

Remove your server from the backup list completely

/changetimezone

Change the timezone for backup scheduling

/changebackupfrequency

Change how often server backups are created (hourly, daily, weekly, monthly, yearly)

/configurebackupcomponents

Choose what gets saved in your server backups (server assets, channels, roles, emojis, stickers)

/separatefiles

Choose if backups should be sent as separate ZIP files or as a single ZIP file

Frequently Asked Questions

Got questions? We got answers, good ones too, If you don't find what you're looking for, join our support server

How often can I schedule backups?

BackupBot allows you to schedule backups hourly, daily, weekly, monthly, or even yearly, You can customize the exact timing based on your needs and the specified timezone.

What happens if my backup file is too large?

If your backup exceeds Discord's file size limit, BackupBot will automatically upload it to secure cloud storage (BackupBot's CDN) and provides you with a download link in your Discord log channel.

Does BackupBot save message history?

No, Currently, BackupBot focuses on server structure, roles, channels, emojis, and server assets, Message history backup is not included in our service due to Discord API limitations and privacy/ToS considerations.

What permissions does BackupBot need?

BackupBot requires some permissions to properly backup all server components, This is necessary to access role configurations, channel settings, and server assets, All permissions are used solely for the backup process and are not used/shared anywhere else.

How secure are my backups?

Your backups are delivered directly to your specified Discord log channel with our CDN's download link, which means you control who has access to them, For larger files stored on our CDN, file links are generated, they're only shared in your log channel, We do not store backup files on our servers after they're successfully sent.

Is BackupBot free to use?

Yes, BackupBot is currently free to use for all Discord servers, We may introduce premium features and paid stuff in the future, but our main backup function will remain accessible to everyone for no money (unless some big company tries to buy it).
